# MOS.RU Water Meter

HACS-интеграция для Home Assistant: автоматически передаёт показания счётчиков воды на портал [mos.ru](https://mos.ru) в заданный день месяца.

Авторизация выполняется через QR-код — пароль вводить не нужно. Показания берутся из любых сенсоров Home Assistant с единицей измерения `м³`.

## Возможности

- Авторизация через QR-код (приложение mos.ru или Госуслуги Москвы), 2FA поддерживается автоматически
- Автоматическая отправка показаний в настраиваемый день месяца (по умолчанию 20-е)
- Поддержка любых HA-сенсоров воды (Waterius, ZigBee, WiFi и т.д.)
- Выбор счётчиков из личного кабинета mos.ru через UI
- Ручная отправка кнопкой в любой момент
- Сенсоры: последние отправленные значения, статус, дата отправки
- Уведомление о необходимости повторной авторизации при истечении сессии

## Установка через HACS

1. В HACS → **Custom repositories** добавьте:
   - URL: `https://github.com/kostinos/hass-mosru-water`
   - Category: `Integration`
2. Найдите **MOS.RU Water Meter** в HACS и установите
3. Перезапустите Home Assistant
4. Настройки → Устройства и службы → **Добавить интеграцию** → `MOS.RU Water Meter`

## Настройка

### Шаг 1 — Реквизиты
- **Код плательщика (ЕПД)** — 10-значный код из квитанции ЖКХ
- **Номер квартиры**

### Шаг 2 — QR-авторизация
В интерфейсе HA отобразится QR-код. Откройте приложение **mos.ru** или **Госуслуги Москвы**, нажмите кнопку сканирования QR и направьте камеру на экран. После сканирования авторизация завершится автоматически.

### Шаг 3 — Выбор счётчиков
Интеграция автоматически получает список ваших счётчиков из mos.ru. Выберите нужные для холодной и горячей воды.

### Шаг 4 — Источники показаний
- Выберите HA-сенсоры, откуда брать показания. Единица измерения должна быть `м³`
- Укажите день месяца для автоматической отправки (1–28)

## Истечение сессии

Если сессия mos.ru истечёт, Home Assistant покажет уведомление «Требуется повторная авторизация». Нажмите на него и снова отсканируйте QR-код — больше ничего делать не нужно.

## Создаваемые сущности

| Сущность | Описание |
|----------|----------|
| `sensor.mosru_water_cold_last` | Последнее переданное значение холодной воды (м³) |
| `sensor.mosru_water_hot_last` | Последнее переданное значение горячей воды (м³) |
| `sensor.mosru_water_status` | Статус последней отправки (`success` / `error` / `pending`) |
| `sensor.mosru_water_submitted_at` | Дата и время последней отправки |
| `button.mosru_water_submit` | Кнопка ручной отправки |

## Зависимости

- `requests >= 2.28`
- `qrcode >= 7.4`

## Отказ от ответственности

Интеграция использует неофициальный API mos.ru. Работоспособность зависит от того, не изменил ли портал свой API.
